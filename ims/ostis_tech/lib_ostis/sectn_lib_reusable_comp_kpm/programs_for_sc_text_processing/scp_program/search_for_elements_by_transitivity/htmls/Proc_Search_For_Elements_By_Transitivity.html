Задачей scp-программы поиска элементов структуры по транзитивности является поиск всех элементов структуры по транзитивности. Первым параметром (in-параметр) является структура, элементы которой необходимо найти. Вторым параметром (in-параметр) является множество, в которое будут заноситься найденные элементы структуры.
