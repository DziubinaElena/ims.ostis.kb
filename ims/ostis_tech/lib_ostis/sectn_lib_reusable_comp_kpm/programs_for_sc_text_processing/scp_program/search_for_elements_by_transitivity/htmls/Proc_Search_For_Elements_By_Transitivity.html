Задачей scp-программы поиска элементов структуры по транзитивности является поиск всех элементов структуры и её подструктур по транзитивности. Первым входным параметром является структура, элементы которой необходимо найти. Вторым входным параметром является множество, в которое будут заноситься найденные элементы структуры.
