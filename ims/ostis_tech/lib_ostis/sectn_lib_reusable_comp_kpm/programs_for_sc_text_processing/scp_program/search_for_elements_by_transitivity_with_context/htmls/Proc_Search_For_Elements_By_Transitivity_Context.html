Задачей scp-программы поиска элементов структуры по транзитивности в рамках заданного контекста является поиск всех элементов структуры по транзитивности с учётом контекста. Первым параметром (in-параметр) является структура, элементы которой необходимо найти. Вторым параметром (in-параметр) является множество, в которое будут заноситься найденные элементы структуры. Третьим параметром (in-параметр) является контекст, в рамках которого будет осуществляться поиск.
