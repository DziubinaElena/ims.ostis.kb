Задачей scp-программы добавления ключевых узлов в ответ scp-прграммы является добавление в ответ scp-программы ключевых узлов, которые не присутсвуют в ответе и экземляры которых принадлежат ответу. Первым входным параметром является множество ключевых узлов, которые необходимо добавить в ответ. Вторым входным паретром является множетсво, в которое осуществляется добавление ключевых узлов.
