Задачей scp-программы подсчёта количества узлов в структуре с заданным контекстом является нахождение количества всех узлов, принадлежащих структуре. Первым параметром (in-параметр) является структура, для которой должно быть найдено количество узлов. Вторым параметром (out-параметр) является sc-ссылка, содержащая число равное количеству узлов, принадлежащих структуре. Третьим параметром (in-параметр) является контекст, в рамках которого осуществляется поиск
