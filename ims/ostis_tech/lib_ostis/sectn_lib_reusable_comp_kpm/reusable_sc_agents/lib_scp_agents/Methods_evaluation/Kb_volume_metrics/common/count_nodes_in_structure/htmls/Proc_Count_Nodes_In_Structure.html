Задачей scp-программы подсчёта количества узлов в структуре является нахождение количества всех узлов, принадлежащих структуре. Первым входным параметром является структура, для которой должно быть найдено количество узлов. Вторым входным параметром является sc-ссылка со знаяением true или false. Если значением sc-ссылки является строка true, то поиск узлов будет осуществляться в рамках структуры и её подструктур. В противном случае поиск осуществляется только в рамках заданной структуры. Выходным параметром является sc-ссылка, содержащая число равное количеству узлов, принадлежащих структуре.
