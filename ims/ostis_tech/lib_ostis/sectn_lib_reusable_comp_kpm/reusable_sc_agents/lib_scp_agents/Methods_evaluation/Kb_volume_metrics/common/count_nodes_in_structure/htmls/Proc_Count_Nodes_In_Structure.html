Задачей scp-программы подсчёта количества sc-узлов в структуре является вычисление количества всех sc-узлов, принадлежащих структуре. Первым параметром (in-параметр) является структура, для которой должно быть найдено количество sc-узлов. Вторым параметром (out-параметр) является sc-ссылка, содержащая число равное количеству sc-узлов, принадлежащих структуре.
