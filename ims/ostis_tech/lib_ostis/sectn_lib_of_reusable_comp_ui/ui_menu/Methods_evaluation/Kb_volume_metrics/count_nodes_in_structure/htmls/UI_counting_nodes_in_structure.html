Команда пользовательского интерфейса для подсчёта количества узлов в структуре предназначена для подсчёта количества узлов, принадлежащих структуре. Единственным входным аргументом является структура, для которой определяется количество узлов. Результатом выполнения команды является значение количества количества узлов, принадлежащих структуре.
